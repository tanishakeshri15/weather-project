# Weather App

A simple Weather App built using HTML, CSS, and JavaScript.

## Features
- Search weather by city name
- Shows temperature
- Shows humidity
- Shows weather conditions

## Technologies Used
- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Run
1. Download the project.
2. Add your OpenWeatherMap API key in `script.js`.
3. Open `index.html` in a browser.
